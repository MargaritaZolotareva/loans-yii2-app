## Запуск

Все изменения находятся в ветке **master**.

❗️ Перед запуском контейнеров необходимо создать файл **.env** в корне проекта, скопировав значения из **.env.example**.

1. Для запуска контейнеров используйте команду:
   * `docker compose up --build -d`

2. Установите зависимости с помощью **Composer**:
   * `composer install`

3. Примените миграции:
   * `php yii migrate`

4. Запуск обработчика заявок (запускать в контейнере backend):
   * `php yii queue/listen`

# Loans API

Простой API для подачи и обработки заявок на займ.

## Технологии
- PHP 8 + Yii2
- PostgreSQL
- Nginx
- Docker Compose

## Эндпоинты

### 1. Подача заявки на займ
**POST** `/requests`  
**Тело запроса:**
```json
{
  "user_id": 1,
  "amount": 3000,
  "term": 30
}
```
Условия: пользователь не может иметь одобренных заявок.

Ответ успешный (HTTP 201):

```json
{
  "result": true,
  "id": 42
}
```
Ответ неуспешный (HTTP 400):

```json
{
  "result": false
}
```
### 2. Обработка заявок

**GET** `/processor?delay=5`

delay – время задержки (сек) для принятия решения по каждой заявке.

Статус заявки устанавливается рандомно: approved (10% шанс) или declined.
У одного пользователя не может быть более одной одобренной заявки.
Нужно эмулировать продолжительное время принятия решения каждой заявки с помощью функции sleep(), в которую в качестве аргумента передать значение delay из запроса. Этот эндпоинт может быть запрошен несколько раз одновременно. Заявки одного пользователя могут обрабатываться параллельно.

Ответ (HTTP 200):

```json
{
  "result": true
}
```
result (boolean): результат выполнения операции.

База данных

```json
host: localhost
port: 5432
db: loans
user: user
password: password
```
